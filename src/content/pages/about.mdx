This is an open source e-commerce store built with everything new in [Next.js 13](https://nextjs.org/).
The project is still in development. You can follow the progress on [GitiHub](https://github.com/bhelpful/mealtime)

## Tech stack used

- [Next.js](https://nextjs.org)
- [Tailwind CSS](https://tailwindcss.com)
- [Clerk Auth](https://clerk.dev)
- [Prisma ORM](https://www.prisma.io/)
- [Contentlayer](https://www.contentlayer.dev)
- [uploadthing](https://uploadthing.com)
- [Stripe](https://stripe.com)
